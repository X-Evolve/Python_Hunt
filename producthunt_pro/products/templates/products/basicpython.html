Basic Definitions



Variables

The variables are the containers that store literals and values in it.


Keywords

The keywords are the reserved words in python or any programming language which has it's own meaning. There are 33 reserved keywords in python.


Built in Functions

Functions are the block of code which performs a certain task and it can be re-used again.
Built in functions are the functions which are already reserved and created in python. The functions which is created by the programmer or the use is called user defined functions.
