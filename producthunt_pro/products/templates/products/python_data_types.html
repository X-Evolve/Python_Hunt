Introduction

In programming, a data type is an attribute of a variable that determines the type of data it can store. It helps the computer understand the nature of the data and how to process it. Understanding data types is crucial in programming as it affects the behavior and operations that can be performed on variables.

Different Data Types

1. Integer (int)

An integer is a whole number without a fractional part. It represents numerical values like 1, 2, -5, etc. 


        # Example code for integers
        x = 5
        y = 10
        sum = x + y
        print("Sum:", sum)

2. Float (float)

A float represents a floating-point number with a fractional part. It is used for decimal values like 3.14, -0.5, etc. 


        # Example code for floats
        pi = 3.14
        radius = 2.5
        area = pi * (radius ** 2)
        print("Area:", area)

3. String (str)

A string is a sequence of characters enclosed in quotes. It is used to represent text data. 


        # Example code for strings
        name = "John Doe"
        greeting = "Hello, " + name
        print(greeting)

4. Boolean (bool)

A boolean represents a logical value that can be either True or False. It is used for conditional statements and logical operations. 


        # Example code for booleans
        is_valid = True
        is_logged_in = False
        print("Is Valid:", is_valid)
        print("Is Logged In:", is_logged_in)

5. List (list)

A list is an ordered collection of items. It can store different types of data and allows for modification of its elements. 


        # Example code for lists
        numbers = [1, 2, 3, 4, 5]
        names = ["John", "Jane", "Alice"]
        print("Numbers:", numbers)
        print("Names:", names)
