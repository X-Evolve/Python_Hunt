Python Sets - Python hunt

Sets in Python have several important properties that make them useful for various tasks. Here, I&#39;ll describe these properties and provide code examples to illustrate each one.

  1. Uniqueness: Sets can only contain unique elements. Adding duplicate elements will not result in duplicates in the set.
my_set = {1, 2, 3, 2, 3}
        print(my_set)  # Output: {1, 2, 3}
  1. Unordered: Sets are unordered, meaning the order of elements is not guaranteed. It will sort elements it's own.
my_set = {3, 1, 2}
        print(my_set)  # Output: {1, 2, 3}
  1. Mutable: Sets are mutable, allowing you to add and remove elements.
my_set = {1, 2, 3}
        my_set.add(4)
        print(my_set)  # Output: {1, 2, 3, 4}
        
        my_set.remove(2)
        print(my_set)  # Output: {1, 3, 4}
  1. Membership Testing: You can check if an element is present in a set using the in operator.
my_set = {1, 2, 3}
        print(2 in my_set)  # Output: True
        print(4 in my_set)  # Output: False
  1. Length of a Set: You can find the number of elements in a set using the len() function.
my_set = {1, 2, 3}
        length = len(my_set)
        print(length)  # Output: 3
  1. Set Operations: Sets support various set operations like union, intersection, and difference. 
set1 = {1, 2, 3}
        set2 = {3, 4, 5}
        
        # Union
        union_set = set1.union(set2)
        print(union_set)  # Output: {1, 2, 3, 4, 5}
        
        # Intersection
        intersection_set = set1.intersection(set2)
        print(intersection_set)  # Output: {3}
        
        # Difference
        difference_set = set1 - set2
        print(difference_set)  # Output: {1, 2}
  1. No Duplicates: When you create a set from an iterable, it automatically removes duplicates. 
my_list = [1, 2, 2, 3, 4, 4]
        my_set = set(my_list)
        print(my_set)  # Output: {1, 2, 3, 4}
  1. Set Literals: Sets can be created using curly braces {}. However, be careful not to use empty curly braces, as they create an empty dictionary.
# Correct set literal
        my_set = {1, 2, 3}
        
        # Incorrect empty dictionary
        empty_dict = {}
  1. Discard Method: The discard() method removes an element from a set if it exists, but it does nothing if the element is not present (no error is raised).
my_set = {1, 2, 3}
        my_set.discard(2)  # Removes 2 from the set
        my_set.discard(4)  # Does nothing since 4 is not in the set

These properties and operations make sets in Python a valuable tool for tasks that involve unique elements, membership testing, and set-based operations.

String in sets

Strings can be elements of sets. However, strings are immutable, so you cannot modify them once they are created. This means you cannot add or remove characters from a string in a set.


            # Creating a set of strings
            string_set = {"apple", "banana", "cherry"}
            
            # Adding a string to the set
            string_set.add("date")
            
            # Checking membership with strings
            print("banana" in string_set) # Output: True
            print("grape" in string_set) # Output: False
            
            # Set operations with strings
            fruits_set = {"apple", "banana", "cherry"}
            berries_set = {"strawberry", "blueberry", "cherry"}
            
            # Union of two string sets
            combined_fruits = fruits_set.union(berries_set)
            print(combined_fruits) # Output: {'apple', 'banana', 'cherry', 'strawberry', 'blueberry'}
            
            # Intersection of two string sets
            common_fruits = fruits_set.intersection(berries_set)
            print(common_fruits) # Output: {'cherry'}
            
            # Difference between two string sets
            unique_fruits = fruits_set - berries_set
            print(unique_fruits) # Output: {'apple', 'banana'}
