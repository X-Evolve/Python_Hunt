List in Python


A list is a versatile and fundamental data structure in Python. It is used to store and manage collections of values, which can be of any data type, such as numbers, strings, or even other lists. Lists have the following important characteristics:

  • Ordered Sequence: Lists contain multiple values that are logically related. These values are stored in a specific order, which means you can access elements based on their positions within the list.
  • Mutable: Lists are mutable, meaning you can change their content after creation. You can add, remove, or modify elements as needed.
  • Indexing: Each element in a list is assigned a numerical position known as an index. Python lists are zero-indexed, meaning the first element is at index 0, the second at index 1, and so on.
  • Slicing: You can extract a portion of a list, known as a slice, by specifying a range of indices. This allows you to work with a subset of the list as needed.
  • Operations: Lists support various operations, such as appending elements, extending lists, multiplying a list by an integer (which replicates the elements), and checking for the presence of an item in the list.
  • Built-in Functions: Python provides built-in functions to make working with lists more convenient. You can find the length of a list using the len() function. You can also use functions like max() and min() to find the largest and smallest elements in a list, respectively.

Iterations & Comprehensions


List Iteration

>>> 2 in [1, 2, 3]
True
>>> for x in [1, 2, 3]:
       print(x, end=' ')
# Iteration (prints x)... 1 2 3

List Comprehensions

>>> res = [c * 4 for c in 'ABCD']
# List comprehensions >>> res
['AAAA', 'BBBB', 'CCCC', 'DDDD']

An expression is functionally equivalent to a for loop that builds up a list of results manually. However, list comprehensions are simpler to code and likely faster to run today.

 

# List comprehension equivalent ...
>>> res = []
>>> for c in 'ABCD':
          res.append(c * 4)
>>> res
['AAAA', 'BBBB', 'CCCC', 'DDDD']

Indexing & Slicing


>>> L = ['abcd', 'ABCD', 'ABCD!']
>>> L[2] # Offsets start at zero
'ABCD!'
>>> L[−2] # Negative: count from right
'ABCD'
>>> L[1:] # Slicing fetches sections
['ABCD', 'ABCD!']
