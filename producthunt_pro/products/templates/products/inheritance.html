Understanding Inheritance in Python


Inheritance is a fundamental concept in Object-Oriented Programming (OOP). It allows you to create new classes based on existing classes.

Parent and Child Classes

Parent Class

The parent class is the original class that defines attributes and methods.

Child Class

The child class is created based on the parent class. It inherits attributes and methods from the parent and can have its own too.

Create a Parent Class

          class Person:
            def __init__(self, firstname, lastname):
            self.firstname = firstname
            self.lastname = lastname

            def printname(self):
              print(self.firstname, self.lastname)

self is the conventional and widely accepted name for the first argument in methods of a Python class, representing the object itself. This object can be used to access its own attributes and methods. firstname and lastname are the parameters passed when creating a new Person object. They represent the first name and last name of the person.

self.firstname and self.lastname are instance variables bound to the object. They store the values of firstname and lastname passed when creating the object.

def printname(self) is a method within the Person class. This method is called to print the person's name. It accesses the instance variables self.firstname and self.lastname to display the name.

If want to create a Person and then execute the printname method. We do it like this:
John = Person("John", "Gillman")

John.printname()

Create a Child Class

If we want the child class to inherit all properties from its parent, we have to use the super keyword.

            class Student(Person):
                def __init__(self, fname, lname):
                    super().__init__(fname, lname)

In this example, the Studentclass has its own constructor with one additional parameter year. It also inherits all properties from its parent class. Additionally, it has a specific method called `welcome`.

            class Student(Person):
                def __init__(self, firstname, lastname, year):
                    super().__init__(fname, lname)
                    self.graduationyear = year

                def welcome(self):
                    print("Welcome", self.firstname, self.lastname, "to the class of", self.graduationyear)

Benefits of Inheritance

  • Reusability: Code from the parent class can be reused in the child class.
  • Structure: Inheritance helps in organizing and structuring your code.
  • Extensibility: You can add new features in child classes without modifying the parent class.
